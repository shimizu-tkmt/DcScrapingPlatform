# DcScrapingPlatform

DC（確定拠出年金）のスクレイピング結果を、多数のユーザーが手軽かつ安全に Google スプレッドシートへ記録できるマルチユーザー向けプラットフォームです。

## プロジェクト構成

- `Doc/`: 設計書、仕様書、導入ガイド
- `Source/`: プログラムのソースコード（Next.js / Python / C# 予定）

## 特徴

- **Google OAuth 2.0**: 独自のパスワード管理を不要にし、セキュアにログイン可能。
- **スプレッドシート自動連携**: スプレッドシート ID の手動入力が不要。
- **暗号化保管**: 確定拠出年金サイトの ID/PASS を AES-256 で強力に暗号化。
- **サーバレス実行**: ブラウザ実行部分（Playwright）を Google Cloud Run 等で低コストに運用。

## ドキュメントの構成

詳細な設計書やガイドラインは `Doc/` ディレクトリを参照してください。
