# 画面設計書の修正完了

ご依頼に基づき、`画面設計書.md` の内容を修正いたしました。本作業はプランニングモードとして実施し、実装コードの変更は含みません。

## 実施日時
2026年3月18日

## 修正内容

### 1. トップページ / ログインの統合
*   トップページをログイン画面として機能するよう再構成。
*   ヒーローセクションに **「メールアドレス」「パスワード」の入力フォーム** を追加。
*   「ログイン」ボタンと「アカウント新規作成」ボタンを配置。

### 2. 連係情報設定画面（旧：初期設定画面）
*   画面名称を **「連係情報設定画面」** に変更。
*   初回登録時の必須入力制限（未完了時のダッシュボードへのリダイレクト）を維持。
*   設定完了後も **ダッシュボードからいつでもアクセス可能** にするよう定義を更新。

### 3. ダッシュボード
*   サイドバーに **「連係情報設定」** へのリンクを追加。

### 4. 画面遷移図（Mermaid）
*   トップページからのログインとアカウント登録。
*   登録後の連係情報設定画面への遷移。
*   ダッシュボードと連係情報設定、アカウント設定間の双方向の遷移。
*   これらを **ラベル付きのノードと矢印** で分かりやすく修正しました。
